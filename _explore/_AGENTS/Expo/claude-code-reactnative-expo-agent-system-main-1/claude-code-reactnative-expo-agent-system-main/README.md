## ğŸ¯ Project-Scoped vs Global

### Agent Precedence

Claude Code checks for agents in this order:
1. **Project agents** (`.claude/agents/` in your project) - HIGHEST priority
2. **Global agents** (`~/.claude/agents/` in home directory) - Lower priority

**Hybrid Approach:** You can have BOTH! Project agents override global agents with the same name.

### When to Use Each

**Project-Scoped (Recommended for Teams):**
- âœ… Team projects (agents shared via git)
- âœ… Project-specific customizations
- âœ… Different agent versions per project
- âœ… Automatic sync for team members

**Global (Personal Use):**
- âœ… Personal projects
- âœ… Consistent agents across all projects
- âœ… Quick setup
- âœ… Single installation for everything
