# SEO Content Quality Rubric - OS 2.0

**Domain:** SEO Content Production
**Version:** 1.0
**Last Updated:** 2025-11-20

---

## Scoring Overview

**Total Score:** 0-100 points across 4 dimensions (25 points each)

**Scoring Interpretation:**
- **90-100:** Excellent - Ready to publish
- **75-89:** Good - Minor improvements needed
- **60-74:** Fair - Significant revisions required
- **0-59:** Poor - Major rework required

---

## Dimension 1: Content Quality (0-25 points)

### 1.1 Writing Quality (0-10 points)

**Excellent (9-10 points):**
- Clear, engaging prose
- Proper grammar and spelling
- Active voice predominant
- Varied sentence structure
- Appropriate reading level for audience
- Natural keyword integration
- Compelling introduction and conclusion
- Logical flow between paragraphs

**Good (7-8 points):**
- Generally clear writing
- Few grammar issues
- Mostly active voice
- Decent flow
- Keywords present

**Fair (5-6 points):**
- Unclear writing
- Multiple grammar errors
- Passive voice heavy
- Choppy flow
- Keyword stuffing

**Poor (0-4 points):**
- Incomprehensible
- Severe grammar issues
- Unnatural language
- No flow

**Common Issues:**
```
‚ùå BAD - Keyword stuffing
"Our SEO services provide SEO optimization for SEO rankings.
Our SEO experts deliver SEO results through SEO strategies."

‚úÖ GOOD - Natural integration
"Our SEO experts develop data-driven strategies to improve your
search rankings. We focus on sustainable, long-term growth."
```

### 1.2 Value & Depth (0-8 points)

**Excellent (7-8 points):**
- Comprehensive coverage of topic
- Original insights or perspectives
- Actionable takeaways
- Real examples/case studies
- Data/statistics cited
- Expert quotes included
- Answers user intent completely
- Goes beyond surface-level

**Good (5-6 points):**
- Good topic coverage
- Some original insights
- Some actionability
- Few examples
- Basic depth

**Fair (3-4 points):**
- Surface-level only
- No original insights
- Generic advice
- No examples
- Shallow

**Poor (0-2 points):**
- No value
- Completely generic
- Doesn't answer intent

### 1.3 Accuracy & Credibility (0-7 points)

**Excellent (6-7 points):**
- All facts verified
- Sources cited appropriately
- Expert review completed
- Recent data (2024-2025)
- No misleading claims
- Author credentials clear
- External links to authoritative sources

**Good (4-5 points):**
- Most facts accurate
- Some sources cited
- Reasonably current
- Minimal misleading info

**Fair (2-3 points):**
- Some inaccuracies
- Few sources
- Outdated information
- Some misleading claims

**Poor (0-1 points):**
- Many inaccuracies
- No sources
- Completely outdated
- Misleading content

## Dimension 2: SEO Technical Optimization (0-25 points)

### 2.1 On-Page SEO (0-10 points)

**Excellent (9-10 points):**
- Title tag optimized (50-60 chars, keyword included)
- Meta description compelling (150-160 chars)
- H1 tag present and optimized (one per page)
- H2-H6 hierarchy logical
- URL structure clean and descriptive
- Image alt text descriptive
- Internal links present (3-5 per 1000 words)
- External links to authority sites

**Good (7-8 points):**
- Title tag present
- Meta description exists
- Header tags used
- URL acceptable
- Some alt text
- Some linking

**Fair (5-6 points):**
- Title tag suboptimal
- Missing meta description
- Poor header structure
- URL issues
- Missing alt text
- Few links

**Poor (0-4 points):**
- No title optimization
- No meta description
- No header tags
- Bad URLs
- No alt text
- No links

**On-Page Checklist:**
```
‚úÖ Title: "Ultimate Guide to SEO Content | Brand Name" (56 chars)
‚úÖ Meta: "Learn proven SEO content strategies to boost rankings..." (155 chars)
‚úÖ H1: "The Ultimate Guide to SEO Content Creation"
‚úÖ H2: "What is SEO Content?"
‚úÖ H2: "Why SEO Content Matters"
‚úÖ H3: "Increased Organic Traffic"
‚úÖ URL: /seo-content-guide (not /page?id=123)
‚úÖ Alt text: "SEO content strategy diagram showing keyword research flow"
‚úÖ Internal link: "Read our keyword research guide"
‚úÖ External link: "According to Google's Search Quality Guidelines..."
```

### 2.2 Keyword Optimization (0-8 points)

**Excellent (7-8 points):**
- Primary keyword in title, H1, first paragraph
- Primary keyword density 0.5-2% (natural)
- Secondary keywords integrated naturally
- LSI keywords present
- Long-tail variations included
- Keywords in alt text (where relevant)
- No keyword stuffing
- Semantic relevance maintained

**Good (5-6 points):**
- Primary keyword present
- Reasonable density
- Some secondary keywords
- Natural usage

**Fair (3-4 points):**
- Keyword barely present
- Poor density (too low/high)
- No secondary keywords
- Unnatural usage

**Poor (0-2 points):**
- Keywords missing
- Keyword stuffing
- Completely unnatural

**Keyword Strategy Example:**
```
Primary: "SEO content strategy"
Secondary: "content marketing SEO", "SEO copywriting"
LSI: "search engine optimization", "organic rankings", "SERP visibility"
Long-tail: "how to develop SEO content strategy", "SEO content strategy for beginners"

‚úÖ Natural usage:
"Developing an effective SEO content strategy requires understanding
your audience and search intent. Modern SEO copywriting balances user
experience with technical optimization to improve organic rankings."
```

### 2.3 Technical SEO (0-7 points)

**Excellent (6-7 points):**
- Mobile-responsive design
- Fast load time (<3s)
- No broken links
- Schema markup implemented
- Canonical tag correct
- No duplicate content
- HTTPS enabled
- XML sitemap updated

**Good (4-5 points):**
- Mobile-responsive
- Acceptable load time
- Few broken links
- Some schema
- Canonical present

**Fair (2-3 points):**
- Mobile issues
- Slow load time
- Multiple broken links
- No schema
- Canonical missing

**Poor (0-1 points):**
- Not mobile-friendly
- Extremely slow
- Many broken links
- No technical optimization

## Dimension 3: User Experience (0-25 points)

### 3.1 Readability & Formatting (0-10 points)

**Excellent (9-10 points):**
- Short paragraphs (2-4 sentences)
- Bullet points and numbered lists used
- Subheadings break up text (every 300 words)
- Bold/italic for emphasis (not overused)
- White space generous
- Font size readable (16px+)
- Line height appropriate (1.5-1.6)
- Reading level appropriate (Flesch score 60-70)

**Good (7-8 points):**
- Reasonable paragraphs
- Some lists
- Subheadings present
- Acceptable formatting
- Decent readability

**Fair (5-6 points):**
- Long paragraphs
- Few lists
- Minimal subheadings
- Dense formatting
- Low readability score

**Poor (0-4 points):**
- Wall of text
- No lists
- No subheadings
- Unreadable

**Formatting Best Practices:**
```
‚úÖ GOOD
## How to Optimize Images for SEO

Image optimization improves page speed and user experience.
Here's how to do it:

- Compress images to <100KB
- Use descriptive file names
- Add alt text for accessibility
- Choose appropriate formats (WebP, JPEG)

Each step contributes to better rankings and faster load times.

‚ùå BAD
How to Optimize Images for SEO: Image optimization is important for
SEO because it improves page speed which is a ranking factor and also
helps with user experience and you should compress your images and use
descriptive file names and add alt text and choose the right formats.
```

### 3.2 Engagement Elements (0-8 points)

**Excellent (7-8 points):**
- Compelling introduction (hook + value proposition)
- Table of contents for long content
- Visual elements (images, infographics, videos)
- Call-to-action clear and relevant
- Interactive elements where appropriate
- Pull quotes or key takeaways highlighted
- Related content suggestions
- Social sharing buttons

**Good (5-6 points):**
- Good introduction
- Some visuals
- CTA present
- Few engagement elements

**Fair (3-4 points):**
- Weak introduction
- Few visuals
- Unclear CTA
- Minimal engagement

**Poor (0-2 points):**
- No hook
- No visuals
- No CTA
- No engagement elements

### 3.3 Content Structure (0-7 points)

**Excellent (6-7 points):**
- Clear introduction (what, why, who for)
- Logical section progression
- Inverted pyramid structure (key info first)
- Supporting details organized well
- Strong conclusion with next steps
- Scannable hierarchy
- Consistent formatting

**Good (4-5 points):**
- Decent introduction
- Reasonable progression
- Some structure
- Acceptable conclusion

**Fair (2-3 points):**
- Weak introduction
- Poor progression
- Minimal structure
- Abrupt ending

**Poor (0-1 points):**
- No structure
- Random organization
- No intro/conclusion

## Dimension 4: Strategic Alignment (0-25 points)

### 4.1 Search Intent Match (0-10 points)

**Excellent (9-10 points):**
- Matches user intent perfectly
- Answers target query directly
- Content type appropriate (informational/transactional/navigational)
- Depth matches SERP expectations
- Satisfies "People Also Ask"
- Addresses related queries
- Better than top 3 SERP results

**Good (7-8 points):**
- Matches intent well
- Mostly answers query
- Appropriate content type
- Comparable to top results

**Fair (5-6 points):**
- Partially matches intent
- Incomplete answer
- Content type mismatch
- Below SERP standards

**Poor (0-4 points):**
- Doesn't match intent
- Fails to answer query
- Wrong content type
- Far below SERP quality

**Intent Types:**
```
Informational: "how to create SEO content"
‚Üí Provide comprehensive guide with steps

Transactional: "buy SEO tools"
‚Üí Product comparison, features, pricing

Navigational: "SEMrush login"
‚Üí Direct link to service

Commercial: "best SEO tools 2025"
‚Üí Comparison, reviews, recommendations
```

### 4.2 Competitiveness (0-8 points)

**Excellent (7-8 points):**
- More comprehensive than competitors
- Unique angles or insights
- Better visuals/formatting
- More recent information
- Superior expertise demonstrated
- Better user experience
- Stronger CTAs
- Content gap filled

**Good (5-6 points):**
- Comparable to competitors
- Some unique elements
- Acceptable quality
- Current information

**Fair (3-4 points):**
- Weaker than competitors
- No unique value
- Below average quality
- Outdated vs competitors

**Poor (0-2 points):**
- Much worse than competitors
- Completely generic
- Poor quality
- Significantly outdated

### 4.3 Business Goals Alignment (0-7 points)

**Excellent (6-7 points):**
- Aligns with business objectives
- Targets right audience segment
- Appropriate funnel stage
- Clear conversion path
- Brand voice consistent
- Supports product/service
- Measurable KPIs defined

**Good (4-5 points):**
- Generally aligned
- Right audience
- Some conversion path
- Brand voice present

**Fair (2-3 points):**
- Weak alignment
- Wrong audience
- Unclear conversion
- Inconsistent voice

**Poor (0-1 points):**
- No alignment
- Wrong audience
- No conversion path
- Off-brand

## Anti-Pattern Library

### 1. Keyword Stuffing
```
‚ùå BAD
"Our SEO agency provides SEO services including SEO optimization,
SEO consulting, and SEO strategy. Contact our SEO experts for
SEO solutions."

‚úÖ GOOD
"Our agency specializes in search optimization, from strategy
development to implementation. Contact our team to discuss your
specific needs."
```

### 2. No Value Introduction
```
‚ùå BAD
"In this article, we will discuss SEO content. SEO content is
important for websites."

‚úÖ GOOD
"73% of marketers say content creation is their top priority, yet
most struggle to rank. This guide reveals the exact framework we
used to increase organic traffic by 312% in 6 months."
```

### 3. Wall of Text
```
‚ùå BAD
[Single 500-word paragraph with no breaks, subheadings, or lists]

‚úÖ GOOD
## What is SEO Content?

SEO content balances user needs with search engine requirements.
Here's what matters most:

- User intent alignment
- Technical optimization
- Content depth
- Engagement signals

Each factor contributes to rankings and conversions.
```

### 4. Missing Technical Elements
```
‚ùå BAD
```

### 5. Generic Content
