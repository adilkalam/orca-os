# Discovery Answers: /record Command

## Q1: Code-level learnings?
**Answer:** NO - human learnings only

Workshop handles decisions/gotchas. Code intelligence is auto-indexed by vibe-sync.

## Q2: Self-improvement integration?
**Answer:** YES - complement loop

Human-triggered /record complements auto-recorded outcomes from pipelines.

## Q3: Interactive confirmation?
**Answer:** YES with --quick bypass

Default confirms, --quick flag skips confirmation.

## Q4: Domain-specific variants?
**Answer:** NO (default accepted)

Single /record with auto-domain detection.

## Q5: Bulk session extraction?
**Answer:** YES - analyze & extract

Can analyze session and extract multiple decisions/gotchas.

---

## Summary

/record will be:
- **Human-focused** - decisions, gotchas, notes (not code patterns)
- **Self-improvement integrated** - complements the automatic outcome recording
- **Interactive by default** - with --quick bypass for power users
- **Domain-agnostic** - auto-detects domain from context
- **Bulk-capable** - can extract multiple items from session history
