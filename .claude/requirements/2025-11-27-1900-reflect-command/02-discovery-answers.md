# Discovery Answers: /reflect Command

## Q1: Signal types?
**Answer:** All signals

Corrections + repeated instructions + explicit feedback.

## Q2: Apply to?
**Answer:** Both systems

CLAUDE.md for rules, Workshop preferences for softer prefs.

## Q3: Trigger?
**Answer:** Manual only

/reflect command triggers capture and analysis. No auto-capture.

## Q4: Infrastructure?
**Answer:** Separate

Completely independent system from agent self-improvement. Different patterns, different outputs.

## Q5: Threshold?
**Answer:** (default accepted) Severity-based

Critical corrections surface quickly; minor preferences accumulate.
